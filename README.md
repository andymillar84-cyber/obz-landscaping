# OBZ Landscaping

Website for OBZ Landscaping — a crazy paving and pool paver specialist based in Teesdale, Victoria, Australia.

Built with React, TypeScript, Vite, Tailwind CSS v4, and Framer Motion.

## Getting Started

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Available Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start the development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Type-check the project |
| `npm run clean` | Remove the `dist` directory |

## Notes

- The hero video and about photo are currently hosted on Google Cloud Storage. Replace these URLs with your own hosted assets before deploying.
- The contact form currently prevents default submission. Wire it up to a form service (e.g. [Formspree](https://formspree.io/)) or a backend endpoint to enable email delivery.
- Social media links in the footer are placeholders — update them with the real Instagram and Facebook URLs.
